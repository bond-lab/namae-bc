Overview

㊔ Welcome to the Japanese Name Database, an interactive tool for exploring trends in Japanese given names (名前 namae).

This site brings together data from three sources covering over a century of Japanese naming practices, and provides tools for searching, comparing and analyzing names by orthography, pronunciation, gender, and year.

Data Sources

Baby Calendar (2008–2022)
Individual baby names with both orthography (kanji/kana) and pronunciation (hiragana), along with location and gender. The original Baby Calendar site has much more information, including pictures — this is a small subset extracted for research purposes.
Heisei Namae Jiten (1989–2009)
Ranked name frequencies covering the Heisei era, with comprehensive frequency distributions (not just the top names). Orthography only.
Meiji Yasuda Life Insurance Rankings (1912–2024)
The longest available series: top 10 written forms from 1912, expanding to top 100 written forms and top 50 pronunciations from 2004.

What You Can Do

  • Search for any name by orthography (書き方) or pronunciation (読み方), or look up individual kanji (漢字).
  • Browse name lists and frequency statistics.
  • Analyze features such as first/last characters, morae, syllables, script type, and name length.
  • Explore phenomena including name diversity over time, genderedness, androgyny, reduplication, irregular readings, and overlap between data sources.
  • View trends by year, with birth-rate normalization.

Use the Settings page to switch between data sources and colour palettes.
