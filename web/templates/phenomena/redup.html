Reduplication in Names

Some Japanese names show reduplication, where part of the name is repeated exactly or with a slight change. This only occurs with the phonological form, unlike Chinese, where typically the orthographic form would also be repeated.

We only have pronuncation in the Baby Calendar data, so this analysis is from there. g

Reduplicated Names

Here we check for all names whose pronunciation has an even number of syllables, where the first half of the pronunciation is the same as the second half.

We only find two syllable female names.
